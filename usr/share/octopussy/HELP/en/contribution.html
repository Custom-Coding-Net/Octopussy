Contribution

You can contribute to Octopussy in many ways:
  • Submit Bug Report/Feature Request
  • Create/Update Service(s)/Report(s)
  • Create new Theme(s)
  • Translate Interface/Documentation into your language
  • Make a donation

Bug Report/Feature Request

You can submit Bug Report/Feature Request directly from 'Support Request' or from the Octopussy GitHub Issues page.

Create/Update Service/Report

You can submit your Service/Report creation/update directly from 'Support Request' or send your files to octo.devel@gmail.com.

Create new Theme

Themes are stored in the AAT Themes directory. (default /usr/share/aat/THEMES/)
Each Theme has its own directory. This directory contains 2 files:
  • style.css (CSS related to Web Interface style)
  • report_style.css (CSS related to HTML Reports generated by Octopussy)

Translate Interface/Documentation

Translations are stored in AAT Translations directory (default /usr/share/aat/Translations/)
Each Language has 2 files, one for the interface and one for the tooltips.
You can send me these files to octo.devel@gmail.com.

Make a donation

You can generously make a donation from the Octopussy Sourceforge Donation page.
