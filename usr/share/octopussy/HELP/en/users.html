Users

Here you can configure your Users.
On this page, you can see all your Users list.

Users List

To disable or enable LDAP Users, go to 'System Configuration'.
Of course, you can only remove Users defined locally in Octopussy.
Every time you will remove a User, a dialog window will ask you to confirm.

Dialog Window Confirmation

On the bottom of this list, you can add a new User.
There is 4 kinds of role for a User:
  • Admin (You can do anything)
  • Read Write (Same as 'Admin' except you don't have access to 'System Configuration' and you can't add/remove Users)
  • Read Only (Same as 'Read Write' except you can't modify anything)
  • Restricted User (Each 'Restricted User' have access to a defined list of Devices, Services, Alerts & Reports)

To define User Restrictions, click on the 'Restricted' link in Users list.
You will be redirected to the 'User Restrictions' page.
