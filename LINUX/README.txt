#
# Software Requirements
#

Before installing Octopussy, be sure to have installed: 
- Apache2
- Apache::ASP
- Perl
- Mod_Perl
- Mysql
- Nscd
- RRDTool & librrd0
- Rsyslog
- htmldoc

and these Perl modules (from CPAN):
- App::Info
- Cache::Cache
- Crypt::PasswdMD5
- Date::Manip
- DBD
- DBD::mysql
- DBI
- JSON
- Linux::Inotify2
- List::MoreUtils
- Locale::Maketext::Simple
- LWP
- Mail::Sender
- Net::FTP
- Net::LDAP
- Net::SCP
- Net::Telnet
- Net::XMPP
- Proc::PID::File
- Proc::ProcessTable
- Readonly
- Sys::CPU
- Unix::Syslog
- URI
- XML::Simple (Take a look http://search.cpan.org/~grantm/XML-Simple-2.18/lib/XML/Simple.pm#ENVIRONMENT for possible performance issues)

Good Luck !
