#
# Software Requirements
#

Before installing Octopussy, be sure to have installed: 
- Apache2
- Apache::ASP
- Perl
- Mod_Perl
- Mysql
- Nscd
- RRDTool & librrd0
- Rsyslog
- htmldoc

and these Perl modules (from CPAN):
- Cache::Cache
- Crypt::PasswdMD5
- Date::Manip
- DBD
- DBD::mysql
- DBI
- JSON
- Linux::Inotify2
- LWP
- Mail::Sender
- Net::FTP
- Net::LDAP
- Net::SCP
- Net::Telnet
- Net::XMPP
- Proc::PID::File
- Proc::ProcessTable
- Sys::CPU
- Unix::Syslog
- URI
- XML::Simple

Good Luck !
